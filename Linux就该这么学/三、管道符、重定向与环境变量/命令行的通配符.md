# Linux系统中的通配符及含义

|   通配符    |      含义      |
| :---------: | :------------: |
|      *      |    任意字符    |
|     ？      |  单个任意字符  |
|    [a-z]    |  单个小写字符  |
|    [A-Z]    |  单个大写字符  |
|    [0-9]    |    单个数字    |
| [[:alpha:]] |    任意字母    |
| [[:upper:]] |  任意大写字母  |
| [[:lower:]] |  任意小写字母  |
| [[:digit:]] |    所有数字    |
| [[:alnum:]] | 任意字母加数字 |
| [[:punct:]] |    标点符号    |

反斜杠（\)：使反斜杠后面的一个变量变为单纯的字符。
单引号（''）：转移其中所有的变量为单层的字符层。
双引号（""）：保留其中的变量属性，不进行转义处理。
反引号（``）：把其中的命令执行后返回结果。
